Melhorias

  • A principal melhoria na questão do site seria a criação de uma versão mobile para o layout, adaptavel para as novas dimensões que atualmente são maioria de público.
  • Ser um layout mais interativo, para que o usuário possa interagir com o conteúdo, passando slides ou algumas demonstrações dinamicas produto.
  • As features estão muito pequenas, poderia ser feito, como já dito acima, uma webdoor passando imagens das features
  • A seção para "Sign Me Up" poderia ser mais destacada, devido ao número grande de informações ela perde visibilidade e possívelmente o usuário não utilizaria. Poderia ser viável uma nova página para contato ou na navbar um botão "GET FREE THEMES".
  • No geral o layout é bem clean e preciso na passagem da informação, só peca em não ser dinamico
