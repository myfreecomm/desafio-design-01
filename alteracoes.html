Alterações de layout

Hum.... Por onde começar....? Acho que a primeira coisa que eu faria seria criar uma logo "visual". Eu acho que somente texto não chama muita atenção, talvez o usuário lembrasse mais da marca se tivesse uma imagem.

Eu mudaria essas imagens do computador/tablet, talvez até tiraria elas desse layout. Mesmo as otimizando, ainda achei que ficaram pesadas e não sei se é a melhor forma de mostrar um layout responsivo sem um texto complementar.

Pensaria num layout responsivo para essa página. Não tem muito conteúdo, não deve ser difícil adaptar.

Com certeza eu tentaria pedir para o designer mudar ou exportar para mim essa textura usada. Eu fiz isso mas não ficou 100% igual.

A lista de "Theme Features" deveria receber mais destaque, é o único lugar com uma descrição do que o tema possui.

Eu deixaria mais explícito quais deveriam ser os efeitos de "hover" e clique nos menus e botões para o usuário ter uma resposta melhor das ações.
