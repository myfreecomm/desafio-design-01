Melhorias de Design

Design Responsivo

É interessante ter um design responsivo para o site, pois assim possibilita um número maior de acessos de diferentes dispositivos, como tablets e smartphones.

Tomei a liberdade de desenvolver um simples layout responsivo para as resoluções de desktop (maiores de 1024px), tablets(de 1023px até 768px) e mobile (de 767px até 320px).

Melhorias de Front-End

Automatizador de tarefas

Durante o desenvolvimento do front-end é importante utilizar um automatizador de tarefas para que haja um aumento de produtividade.

Durante o desenvolvimento desse layout, utilizei o Grunt para fazer o pré-processamento do SASS e para minificar o arquivo JS.

Pré-processador CSS

É interessante também o uso de pré-processadores css tendo em vista o aumento de produtividade, escrevendo mais em menos tempo.

Como eu já havia mencionado, utilizei o SASS para poder escrever o estilo desse layout.

O CSS foi compilado no caminho build/css/style.css. Já o arquivo de origem do SASS é src/scss/estilo.scss.

Minificador

É interessante minificar os arquivos CSS e JS para ajudar na performace do site.

Para o desafio, utilizei o Grunt para minificar o arquivo JS na pasta build/js/script.min.js. O arquivo JS de origem está na pasta src/js/script.js.

Compressor de imagem

Para o desafio de design, utilizei um compressor para reduzir o tamanho das imagens sem apresentar uma perda de qualidade e assim melhorar a performace do site.

Após comprimir as imagens, pude notar houve uma redução de 65% no peso delas, assim fazendo com que a página tenha um carregamento mais rápido.
